# Sistema de Gerenciamento de Usuários e Produtos

Projeto da disciplina de Engenharia de Software - 2° Desenvolvimento de Sistemas

## Como Executar

```bash
cd ProjetoFinal\src
python main.py
```

## O Que Funciona

- Adicionar usuários com validação
- Remover usuários
- Adicionar produtos
- Remover produtos
- Interface em abas

## Validações

**Usuários:**
- CPF único (11 dígitos)
- Email obrigatório
- Todos os campos preenchidos

**Produtos:**
- Nome não pode estar vazio
- Preço deve ser maior que zero
- Quantidade não pode ser negativa

## Estrutura

```
ProjetoFinal/
├── src/
│   ├── main.py           # Ponto de entrada
│   ├── modelo.py         # Classes Usuario e Produto
│   ├── repositorio.py    # DAO - armazena dados
│   ├── controlador.py    # Lógica de negócio
│   └── visao.py          # Interface Tkinter
└── docs/                 # Documentação
```

## Tecnologias

- Python 3.8+
- Tkinter (built-in)
- Padrão MVC
